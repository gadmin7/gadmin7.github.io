  • Closed Domain Event Extraction On RAMS Dataset

    This project leverages BERT-based models for effective event extraction on the RAMS dataset, focusing on identifying events and their arguments. We address both intra-sentence and cross-sentence argument extraction by converting event trigger and argument detection into standard BIO tagging (token classification) tasks.

    Completed: June 2024
    GitHub Repo

  • Real Time Translation to Remove Communication Barrier (ISL Recognition)

    A comprehensive solution designed to dismantle communication barriers by integrating three core components: real-time language translation, Indian Sign Language (ISL) recognition, and NLP-based English-to-ISL conversion. Our approach achieved high accuracy on ISL recognition by training on two custom-created datasets featuring video and hand landmark data.

    Date: June 2021
