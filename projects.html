Closed Domain Event Extraction On RAMS Dataset

This project leverages deep learning for event extraction on the RAMS dataset, addressing both intra-sentence and cross-sentence argument extraction. By converting event trigger and argument detection into token classification tasks using BIO tagging, we apply BERT-based models for effective event extraction.

Real Time Translation To Remove Communication Barrier

We develop a one-stop solution for communication barriers, integrating language translation, Indian Sign Language (ISL) recognition, and NLP-based English-to-ISL conversion into a single web application. Our approach achieves high accuracy on ISL recognition using two custom-created datasets, including video and hand landmark data.

Date: June 2021

Object Detection In Indian Food Platters Using Transfer Learning With YOLOv4

We present IndianFood10, a comprehensively labeled dataset for object detection in Indian food dishes, overcoming challenges like non-distinct boundaries and high intra-class variation. Using YOLOv4 with transfer learning, our model achieves a 91.8% mAP and 0.90 f1-score, with an extension dataset, IndianFood20, adding 10 more classes.

Date: May 2022
